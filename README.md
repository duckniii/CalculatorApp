# 🧮 CalculatorApp

A modern, all-in-one calculation and conversion tool for Android. This app combines a powerful scientific engine with essential daily utilities like currency and unit converters.

## 📲 Download
The latest version is available right here on GitHub. 
**[👉 Download the latest APK](https://github.com/YuuriEnjoyer/CalculatorApp/releases)**

---

## 🚀 Key Features

### 🔢 Mathematics
- **Standard & Scientific:** Full support for basic arithmetic plus trigonometry (sin, cos, tan), logarithms, and constants.
- **Fractions:** Calculate with fractions and mixed numbers effortlessly.
- **History:** Keep track of your previous calculations so you never lose your work.

### 🌎 Converters
- **Live Currency:** Real-time exchange rates for global currencies (requires internet).
- **Units:** Over 10+ categories including Weight, Temperature, Speed, Volume, and Data Storage.
- **Time Zones:** Easily calculate time differences between cities worldwide.

### 💰 Finance & Daily Use
- **Tip & Bill Splitter:** Instantly calculate tips and split the total among any number of people.

---

## 🛠 Technical Details
- **Platform:** Android (minSdk 24)
- **Language:** Kotlin
- **Architecture:** MVVM with ViewModels
- **Networking:** Retrofit/OkHttp for fetching currency data
